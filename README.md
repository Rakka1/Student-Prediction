# Student-Prediction
A machine learning model that predicts grades
The data set was obtained from kaggle at, https://www.kaggle.com/code/ahmedwaelnasef/student-higher-education-ml-algorithms-with-gradio

The model uses features age, study time, sex, Guardian, failures, internet access and a few others to predict a students G3(GRADE).

G3 Model - Train Score: 0.89, Test Score: 0.12
